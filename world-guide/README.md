# HPMOR World Guide

A comprehensive reference guide for Harry Potter and the Methods of Rationality, designed for fanfiction writers and fans who want quick, accurate information about the HPMOR universe.

## What This Is

This world guide extracts and organizes all world-building information from the 124 chapters of HPMOR (~636K words) into structured, searchable documents. Each entry includes:

- **Direct quotes** from the source text
- **Chapter citations** for every claim `[Ch. 42]`
- **Confidence levels** to distinguish fact from inference
- **Cross-references** to related concepts
- **Writer-practical guidance** for creating fanfiction

## Confidence Level System

Throughout this guide, claims are tagged with confidence levels:

- **[EXPLICIT]** - Directly stated in the text, often with a direct quote
- **[IMPLIED]** - Strongly suggested by the text, clear from context
- **[INFERRED]** - Reasonable conclusion from multiple data points
- **[AMBIGUOUS]** - Deliberately unclear or contradictory in source material

This system helps you distinguish between established canon and open-for-interpretation elements.

## Citation Format

All claims include chapter citations:
- `[Ch. 42]` - Single chapter
- `[Ch. 42-45]` - Range of chapters
- `[Ch. 7, 15, 23]` - Multiple specific chapters
